# LLM Inference Platform

Enterprise-grade LLM inference gateway with intelligent routing, model orchestration, and production observability.

## Features

- Multi-model routing
- GPU scheduling
- Redis caching
- PostgreSQL persistence
- Prometheus metrics
- Docker & Kubernetes ready

## Requirements

- Python 3.11+
- Docker & Docker Compose
- GPU (optional)

## Setup

Clone the repository:

```bash
git clone <repo-url>
cd llm-inference-platform
